# PAXLAB Subs - DEV2.11.8

Module local de génération et prévisualisation de sous-titres PAXLAB.

DEV2.11.8 est une passe UI/UX uniquement, basée sur DEV2.11.7. Le moteur, la transcription, le CTC auto q8, les exports SRT/VTT/JSON et le comportement de calage restent inchangés. L'interface est restructurée en trois zones : Source, Preview, Timeline. Les réglages techniques et diagnostics sont repliés dans un panneau avancé pour retrouver une façade simple et premium.

## Cloudflare Pages

Output directory : `dist`
Root directory : vide
